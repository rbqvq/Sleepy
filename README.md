# Sleepy (Go Edition)

一个 ~~用于视奸~~ 查看个人的在线状态 (以及正在使用软件) 的网站，让他人能知道你不在而不是故意吊 TA

> 使用 Golang 重写的全新架构, 采用 WebSocket + gRPC! (不兼容其他版本)
